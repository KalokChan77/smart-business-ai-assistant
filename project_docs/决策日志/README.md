# 项目决策日志索引

决策日志用于记录项目开发过程中“为什么这样做”，为技术复盘、项目报告、答辩材料和本科生实习日记提供依据。

## 编号规则

文件名格式：`DEC-YYYYMMDD-NNN-简短标题.md`

- `YYYYMMDD`：决策日期；
- `NNN`：当天决策序号，从 `001` 开始；
- 一个日志只记录一个核心决策；
- 后续发生变化时，不删除旧日志，应新增日志并标记替代关系。

## 日志状态

- `提议`：仍在讨论；
- `已接受`：已经采用；
- `已替代`：被新决策替代；
- `已废弃`：不再采用。
