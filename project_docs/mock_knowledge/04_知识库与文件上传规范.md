# 知识库与文件上传规范

> 教学模拟文档。

## 支持格式

教学项目默认支持 PDF、DOCX 和 TXT。旧版 DOC、压缩包、可执行文件和带宏文档不进入自动索引流程。

## 文件限制

单个模拟文件建议不超过 20 MB。上传后状态依次为 pending、processing、completed 或 failed。只有 completed 状态的文档可以参与知识检索。

## 文档更新

管理员上传同名新版本时，应创建新版本记录。新版本索引成功后再停用旧版本，避免知识库在处理中暂时不可用。

## 引用要求

知识库回答必须尽量返回文档名称和相关片段。检索不到可靠内容时，应回答“当前知识库中没有找到足够依据”，不能编造制度、价格或联系方式。
