# 订单、退款与售后政策

> 教学模拟文档，不构成真实合同或法律承诺。

## 订单状态

订单状态包括：待支付、已支付、处理中、已完成、退款审核中、已退款和已取消。

## 订单查询

订单查询必须提供订单编号。系统只允许用户查询本人订单；客服人员查询他人订单时必须具有客服权限并记录审计日志。

## 模拟退款规则

用户在支付后 7 个自然日内、服务尚未激活且未产生 AI 调用记录时，可以提交全额退款申请。服务已经激活或已经使用 AI 调用额度时，退款申请进入人工审核，AI 不能直接承诺退款金额或审核结果。

## 退款处理时间

退款申请提交后预计在 3 个工作日内完成审核。审核通过后，模拟退款在 5～10 个工作日内原路退回。

## 不予自动处理的情况

以下情况必须转人工处理：订单编号不存在、订单归属不一致、支付金额异常、超过退款期限、用户要求修改收款账户或存在疑似欺诈行为。
